Steps to create a Website for a Company


    Research Phase:

  1. Learn about the requirements of the website.
  2. understand the "must haves" and the "could haves" of the website.

    3. Design Phase:

  3. design the user interfaces and mock up views or models for the website.

    4. Development Phase:

  4. implement the "must haves" before the "could haves" of the website
  5. design the final website and test it.
